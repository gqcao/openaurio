# openaurio — 有灵魂的语言学习伙伴

## 一句话介绍

**openaurio是一个有灵魂的AI语言学习伙伴，通过情感连接和个性化对话帮助移民学习语言。**

---

## 项目背景

在瑞典，每年有数万移民等待SFI（瑞典语作为第二语言）课程，等待期长达数月。期间他们缺乏语言练习机会，难以融入当地社会。传统语言学习应用虽然功能丰富，但缺乏情感连接，用户留存率低（<10%）。

**核心痛点：**
- 📋 SFI课程等待名单过长
- 🤖 传统AI应用缺乏"人情味"
- 💔 用户缺乏情感连接，容易放弃
- 🗣️ 口语练习机会稀缺

---

## 解决方案：有灵魂的语言伙伴

**openaurio从瑞典语学习起步，但我们的愿景将扩展到英语、西班牙语、德语、法语、日语等更多语言。每种语言都有独特的角色和真实的文化背景。**

**认识Vera、Lars、Emma等角色——每个都是拥有完整人生、个性和教学风格的真实角色。**

## 核心特性

### 1. 🎭 有灵魂的角色
- **完整人格设定**：年龄、职业、家庭、爱好、住所
- **情绪智能**：感知用户心情并给予情感回应
- **个人记忆**：记住用户的名字、进度、生活故事
- **角色一致性**：每个伙伴都有自己的生活、故事、口头禅

### 2. 🗣️ 语音对话
- **语音转文字**：用户发送语音，伙伴理解并回复
- **文字转语音**：伙伴用温暖自然的声音回应
- **真实口语练习**：不是选择题，是真正的对话

### 3. 📚 场景式学习
- ☕ **咖啡馆** — 点餐和瑞典咖啡文化
- 🛒 **超市购物** — 日常生活购物
- 🏠 **租房咨询** — 瑞典租房和住房
- 每个场景包含文化提示和关键短语

### 4. 🎮 游戏化系统
- **XP经验值和等级**：A1 → A2 → B1 → B2 进阶
- **连续学习天数追踪**：每日练习动力
- **9个成就徽章**：里程碑解锁
- **进度可视化**：见证成长

### 5. 🇸🇪 SFI课程对齐
- 内容与瑞典官方SFI课程对齐
- 覆盖A1到B1级别
- 适合SFI等待期间预习

### 6. 🔧 智能工具
- **天气集成**："哥德堡今天天气怎么样？"
- **网页搜索**：时事新闻和文化问题
- **上下文感知**：伙伴知道何时使用工具

1. **"灵魂"而非"功能"** — 不是更多功能，而是更深连接
