# openaurio - 有灵魂的语言学习伙伴

## 一句话介绍

**openaurio是一个有灵魂的AI语言学习伙伴，通过情感连接和个性化对话帮助移民学习瑞典语。**

---

## 项目背景

在瑞典，每年有数万移民等待SFI（瑞典语作为第二语言）课程，等待期长达数月。期间他们缺乏语言练习机会，难以融入当地社会。传统语言学习应用虽然功能丰富，但缺乏情感连接，用户留存率低（<10%）。

**核心痛点：**
- 📋 SFI课程等待名单过长
- 🤖 传统AI应用缺乏"人情味"
- 💔 用户缺乏情感连接，容易放弃
- 🗣️ 口语练习机会稀缺

---

## 解决方案：Vera

**Vera是一位62岁的退休瑞典语老师，住在哥德堡。她不只是教你语言，她关心你的生活。**

### 核心特性

**1. 🎭 有灵魂的角色**
- 完整人格设定：退休老师、有猫、有花园、有故事
- 情绪智能：感知用户心情并给予情感回应
- 个人记忆：记住用户的名字、进度、生活故事

**2. 🗣️ 语音对话**
- 语音转文字：用户发送语音，Vera理解并回复
- 文字转语音：Vera用温暖的瑞典语声音回应
- 真实口语练习：不是选择题，是真正的对话

**3. 📚 场景式学习**
- ☕ 咖啡馆点餐（Fika）
- 🛒 超市购物
- 🏠 租房咨询
- 每个场景包含文化提示和关键短语

**4. 🎮 游戏化系统**
- XP经验值和等级系统（A1→A2→B1→B2）
- 连续学习天数追踪
- 9个成就徽章
- 进度可视化

**5. 🇸🇪 SFI课程对齐**
- 内容与瑞典官方SFI课程对齐
- 覆盖A1到B1级别
- 适合SFI等待期间预习

## 创新点

1. **"灵魂"而非"功能"** — 不是更多功能，而是更深连接
2. **情绪智能** — AI感知用户心情并调整回应
3. **角色一致性** — Vera有自己的生活、故事、爱好
4. **语音优先** — 强调口语练习而非选择题
5. **社会价值** — 帮助移民融入瑞典社会
